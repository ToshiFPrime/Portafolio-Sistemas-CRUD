Sobre Mí

Soy estudiante de Ingeniería en Sistemas cursando el quinto semestre. Mi enfoque profesional se centra en la arquitectura de hardware y la optimización de sistemas críticos. Poseo una sólida base en resolución de problemas complejos, desde la capa física de red hasta la implementación de lógica de software para automatización.

Especialidades Técnicas

🛠️ Hardware y Periféricos

Realizo diagnósticos avanzados a nivel de componentes. Especialista en mantenimiento preventivo y correctivo. Incluyo servicios de alta precisión como la reparación de mandos de consolas (reemplazo de joysticks para eliminar Stick Drift) y limpieza ultrasónica de componentes para extender la vida útil del hardware bajo estrés térmico.

🌐 Infraestructura de Redes

Diseño, implementación y administración de redes LAN/WLAN. Configuración de protocolos de enrutamiento y segmentación de red mediante VLANs. Utilizo herramientas de simulación como Cisco Packet Tracer para validar topologías antes de su despliegue físico, garantizando la seguridad y eficiencia en la transmisión de paquetes.

🎮 Optimización y Gaming

Especialista en "Low Latency Gaming". Configuración de sistemas operativos para minimizar procesos en segundo plano y optimizar el frame-time. Gestión de servidores dedicados para plataformas como Minecraft, aplicando mods de optimización de memoria (Forge/Fabric) para asegurar estabilidad en entornos multijugador masivos.

💻 Desarrollo y Scripting

Automatización de tareas repetitivas mediante el lenguaje Python. Desarrollo de scripts para el procesamiento de archivos, análisis de datos con librerías especializadas y creación de pequeñas herramientas de escritorio que optimizan el flujo de trabajo técnico diario.
