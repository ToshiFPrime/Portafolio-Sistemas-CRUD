## 🚀 Descripción del Proyecto
Este proyecto es una aplicación web integral que combina un portafolio profesional (Front-end) con un módulo administrativo de gestión de servicios (Back-end simulado). Implementa un CRUD funcional utilizando tecnologías web estándar.

## 🛠️ Tecnologías Utilizadas
* **HTML5:** Estructura semántica y formularios mixtos.
* **CSS3:** Diseño responsivo, CSS Grid, Flexbox y Variables Nativas para el **Modo Oscuro**.
* **JavaScript (Vanilla):** Lógica de interacción, validación de datos y manipulación dinámica del DOM.
* **SQL:** Estructura de base de datos relacional para el almacenamiento de registros.
